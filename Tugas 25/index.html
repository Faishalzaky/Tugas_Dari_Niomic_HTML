Mengenal Sematic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada diwebsite kita, tag article bisa kita gunakan dengan element header dan footer. Karena article type block, artinya element ini memiliki lebar browser jika kita tidak menentukan lebarnya

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside dibagian bawah postingan untuk menjelaskan identitas penulis artikel. Kerena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
